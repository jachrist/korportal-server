API-spesifikasjon

Request/response-format for alle Power Automate-endepunkter

Generelt responseformat

Alle GET-flows returnerer:

{
  "success": true,
  "data": [ ... ],
  "count": 10
}

POST-flows returnerer:

{
  "success": true,
  "message": "Beskrivelse av resultat",
  "id": "eventuell-ny-id"
}

Ved feil:

{
  "success": false,
  "error": "Feilmelding som kan vises til bruker"
}

Autentisering

POST /auth/send-kode

Variabel: POWER_AUTOMATE_AUTH_SEND_CODE_URL

Request:

{
  "email": "medlem@example.com"
}

Response (success):

{
  "success": true,
  "message": "Kode sendt til medlem@example.com"
}

Response (ukjent e-post):

{
  "success": false,
  "error": "E-postadressen er ikke registrert som medlem"
}

POST /auth/verifiser-kode

Variabel: POWER_AUTOMATE_AUTH_VERIFY_CODE_URL

Request:

{
  "email": "medlem@example.com",
  "code": "123456"
}

Response (success):

{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "member": {
    "id": 42,
    "name": "Ola Nordmann",
    "email": "medlem@example.com",
    "role": "medlem",
    "voice": "tenor 1"
  }
}

    {
      "id": 1,

FeltTypeBeskrivelse
titlestringMenyelement-navn
urlstringLenke
iconstring?Ikon (emoji eller null)
ordernumberSorteringsrekkefolge
rolestringMinimum rolle: anonym, medlem, styre, admin
hideWhenLoggedInbooleanSkjul nar bruker er innlogget
isLogoutbooleanOm dette er utloggingsknapp

Ovelse

GET /ovelse/program

Variabel: POWER_AUTOMATE_PRACTICE_URL

{
  "success": true,
  "data": {
    "title": "Vaarprogram 2026",
    "voice": "tutti",
    "baseUrls": {
      "pdf": "https://storage.blob.core.windows.net/sanger/",
      "audio": "https://storage.blob.core.windows.net/sanger/"
    },
    "notes": [
      {
        "id": "Sanctus",
        "noteTitle": "Sanctus",
        "pdfFilename": "Sanctus - Ola Gjeilo.pdf",
        "audio": {
          "sopran 1": "Sanctus - Sopran 1.mp3",
          "tutti": "Sanctus - Tutti.mp3"
        },
        "pageTurns": [
          { "time": 45.2, "page": 2 },
          { "time": 92.8, "page": 3 }
        ]
      }
    ]
  }
}
FeltTypeBeskrivelse
baseUrls.pdfstringBasis-URL for PDF-filer
baseUrls.audiostringBasis-URL for lydfiler
notes[].pdfFilenamestringKombineres med baseUrls.pdf
notes[].audioobjectNokkel = stemmegruppe, verdi = filnavn
notes[].pageTurns[].timenumberSekunder for sideskift
notes[].pageTurns[].pagenumberSidenummer (1-indeksert)

{

    { "time": 92.8, "page": 3 }
