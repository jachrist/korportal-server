Power Automate navnestandard

Format: [Prosjekt] - [Modul] - [Handling] - [Beskrivelse]

Prefix: Alle flows starter med Utsikten -

Utsikten - Noter - GET - Bibliotek GET Henter notebiblioteket
Utsikten - Nedlasting - GET - Filer GET Henter nedlastbare filer
Utsikten - Konserter - GET - Liste GET Henter konserter
Utsikten - Konserter - POST - Billett POST Bestiller billetter
Utsikten - Billetter - GET - Ubetalte GET Henter ubetalte reservasjoner
Utsikten - Billetter - POST - MarkerBetalt POST Markerer som betalt
Utsikten - Billetter - POST - Slett POST Sletter ubetalte reservasjoner

Responseformat

GET-flows returnerer:

{ success: true, data: [...], count: N }

POST-flows returnerer:

{ success: true, message: "...", id: "..." }

Feil:

{ success: false, error: "Feilmelding" }
